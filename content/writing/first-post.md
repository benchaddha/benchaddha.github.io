A simple process beats a complicated process:

1. Start with a falsifiable question.
2. Build the smallest experiment that can fail.
3. Write as you go.

```python
question = "Does feature X improve robustness?"
result = run_experiment(question)
print(result.summary())
```

Good notes today become a paper draft tomorrow.
